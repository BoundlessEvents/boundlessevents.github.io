I (Emareks) created this site for anyone to be able to find a list of events going on in the boundless community.
I realize that not everyone uses discord or the forums so I started keeping track of events I could attend with my guild.

I attempt to add events I see posted on the forums on a public calendar, but I realize some people might want to maintain their own.
Simply create a new public calendar and create a new issue on github and provide the public url link.
